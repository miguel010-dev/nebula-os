# Sobre

O **Nebula OS** é um sistema operacional baseado na web desenvolvido por **Miguel Viana Oliveira**.

O projeto nasceu com o objetivo de estudar engenharia de software em larga escala através da construção de um ecossistema completo, onde cada componente conversa entre si de maneira organizada.

Mais do que criar uma interface semelhante a um sistema operacional, o Nebula OS busca reproduzir conceitos reais encontrados em sistemas modernos, como gerenciamento de aplicações, autenticação, armazenamento de arquivos, APIs, organização modular e experiência do usuário.

---

# Filosofia

O Nebula OS foi criado seguindo quatro pilares principais.

- Simplicidade
- Performance
- Organização
- Escalabilidade

Cada funcionalidade do sistema é desenvolvida pensando na facilidade de manutenção e expansão futura.
